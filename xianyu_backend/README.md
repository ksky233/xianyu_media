## 短剧分类
### 一、按**核心题材/主题**划分（最常见的内容分类）
这一维度聚焦短剧的核心故事主题或背景，是观众最易感知的分类方式：  
- **爱情**：以情感发展为核心，细分包括甜宠（如《双世宠妃》短剧版）、虐恋（误会/分离为主线）、暗恋、破镜重圆等，侧重男女主情感互动。  
- **悬疑**：以谜题、案件或未知事件推动剧情，细分包括探案（如刑侦破案）、惊悚悬疑（含灵异元素）、心理悬疑（角色心理博弈）等，强调“未知感”和“推理过程”。  
- **古装**：以古代社会为背景（真实或架空），涵盖宫廷权谋（如皇子夺嫡）、江湖武侠（侠客恩怨）、古代生活（市井/贵族日常）等，服化道贴合古代风格。  
- **穿越**：以“时空穿梭”为核心设定，包括现代穿古代（最常见）、古代穿现代、跨时空互动等，剧情多围绕“适应新时空”或“改变命运”展开。  
- **都市职场**：聚焦现代城市生活，如职场竞争（如《初入职场的我们》短剧版）、行业揭秘（律师、设计师等）、都市男女的生活压力等，贴近现实。  
- **家庭伦理**：围绕家庭关系展开，如婆媳矛盾、夫妻婚姻、亲子教育、家产纠纷等，剧情多带现实冲突和情感张力。  
- **校园青春**：以学生时代为背景，如初恋、校园霸凌、高考压力、社团生活等，基调多清新、怀旧或励志。  
- **奇幻仙侠**：含超自然元素，如神仙、妖魔、修仙、法术等，剧情多为“仙凡恋”“三界大战”“升级打怪”，画面偏唯美或热血。  
- **乡村生活**：展现农村风貌，如返乡创业、乡村爱情、扶贫故事、邻里互助等，风格淳朴接地气，部分带治愈感。  


### 二、按**核心剧情设定/元素**划分（侧重故事的关键设定）
这类分类以“剧情的核心机制或特色元素”为标签，常与题材交叉融合：  
- **重生逆袭**：主角带着前世记忆重来，目标多为复仇、弥补遗憾或逆袭人生（如“重生后从弃女变女王”），爽感强。  
- **系统流**：主角绑定“系统”（虚拟任务助手），通过完成任务获得奖励（技能、财富等），如“系统逼我当首富”“系统指导我谈恋爱”。  
- **甜宠/虐恋**（爱情题材的细分）：甜宠主打“高糖无虐”，如霸总独宠、双向奔赴；虐恋主打“情感波折”，如爱而不得、误会重重。  
- **悬疑推理**（悬疑题材的细分）：更强调逻辑推理，如密室案件、连环杀人案、凶手侧写等，主角多为侦探或高智商普通人。  
- **科幻未来**：以科技或未来世界为背景，如人工智能（AI与人类的关系）、时空折叠、星际移民等，剧情带科幻想象。  


### 三、按**风格/基调**划分（侧重整体叙事氛围）
- **搞笑喜剧**：以幽默为核心，通过夸张台词、无厘头剧情或反差人设制造笑点，如职场吐槽、家庭闹剧、谐音梗密集的短剧。  
- **治愈温情**：基调舒缓温暖，多讲述平凡人的小确幸（如“街角咖啡馆的暖心故事”）、亲情友情的美好，或主角自我和解的过程。  
- **爽剧**：主打“解压感”，主角快速解决冲突、打脸反派、一路开挂（如“废柴逆袭成大佬”“被欺负后强势反击”），节奏快、冲突直接。  
- **单元剧**：单集或几集一个独立故事，主题多样（如“都市奇谈”“人生百态”），剧情互不关联，适合碎片化观看。
